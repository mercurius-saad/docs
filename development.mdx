## Preview locally

<Steps>
<Step title="Install the Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="Run the dev server">

```bash
cd docs/
mint dev
```

A local preview will be available at `http://localhost:3000`.
</Step>
</Steps>

## Deployment

This docs site is hosted on [Mintlify](https://mintlify.com) and auto-deploys from the `main` branch of the `mercurius-saad/docs` GitHub repo. Push to `main` and Mintlify will rebuild within ~60 seconds.

## Adding pages

1. Create a new `.mdx` file in the appropriate directory
2. Add the page path to `docs.json` under the correct tab/group
3. Use frontmatter for title and description:

```mdx
---
title: "Page Title"
description: "Brief description"
---
```

## Navigation structure

The site is organized into tabs defined in `docs.json`:

| Tab | Content |
|-----|---------|
| **Overview** | Landing page, tech stack, getting started guides |
| **Architecture** | Codebase structure, auth, database, models, workflows, backend systems |
| **Integrations** | Stripe, Twilio, Zoom/Daily, Google, identity verification, AWS |
| **API Reference** | All backend endpoints with middleware and params |
| **Frontend** | Vue 2 client — components, routes, API integration, deployment |
| **DevOps** | Testing, Docker, CI/CD, AWS infrastructure, gotchas |
| **AI Tools** | Cursor, Claude Code, Windsurf setup guides |

## Useful Mintlify components

| Component | Usage |
|-----------|-------|
| `<Note>` | Informational callout |
| `<Warning>` | Warning callout |
| `<Card>` / `<CardGroup>` | Linked cards for navigation |
| `<Steps>` / `<Step>` | Numbered step-by-step instructions |
| `<Tabs>` / `<Tab>` | Tabbed content |
| `<Accordion>` / `<AccordionGroup>` | Collapsible sections |
